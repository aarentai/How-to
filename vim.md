## Opening a New File
```
vim [file] # open file
i          # switch to insert mode
...        # enter text
[Esc]      # switch back to command mode
:wq        # write file and exit vim
```

## ... or editing the Existing File
```
vim [file] # open file
h/j/k/l    # left/down/up/right
nG         # 10G goes to line 10
[Esc]      # switch back to command mode
:wq        # write file and exit vim
```
